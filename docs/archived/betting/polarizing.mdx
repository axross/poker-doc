# ポラライズ

## オーバーサイズベット

- 勝利を確信していて（ナッツハンドなど）、相手が一定以上の強さのメイドハンドである時、適切なベットサイズ（=コールしてもらえる最大額）は思っているより高い
  - これは心理的な作用が強く、人はその時点での ROI ではなく埋没必要を考慮してしまいがちであるから。ROI に関係するのは（それまでに積み上げた）ポット額であり、それまでに投資したベット額ではない
  - 仮に、自分がそれまでにプリフロップで 9BB、フロップで 9BB、ターンで 15BB のベットをしていて、リバーで相手に 40BB のベットをされたとする。詳細な計算を省いてポットが 110BB ほどとした時、考えるべきは自分のハンドの期待値が 40BB ぶんあるかどうかである
  - ここで埋没費用を考慮してしまうと、既にそれまでにベットした 33BB の上に 40BB を加えることが正当化されるかどうかというおかしな問いになってしまう。考えればわかることなのに、実際のテーブル上ではプレッシャーのせいでこの間違いを犯しやすい
