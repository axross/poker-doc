# バリューベット

コールしてもらうことを目的として行うベットを**バリューベット**と言う。

## バリューターゲット

相手のハンドレンジのうち、自分のハンドが勝っていてバリューベットの対象となるサブレンジを**バリューターゲット**と言う。

例えば、ボードがドライな Ad2h8cJh4c で相手がプリフロップから一貫して強い主張をしているとき、 8s8d を持っているプレイヤーにとって、バリューターゲットは:

- AxKx、AxJx、Ax8h、Ax2x、Ax4x、2x2x
  - **強いバリューターゲット。**
  - 相手はこちらの強いアクションに抵抗できるだけの力があるので、オーバーポットのベットをしてもコールしてもらえる可能性が高い。
- AxQx、AxTx、Ax9x、Ax7x、Ax6x、Ax5x、Ax3x
  - **弱いバリューターゲット。**
  - これらはオーバーポットのベットにコールするには比較的弱く、フォールドされてしまう可能性もある。ただしベット額が小さければコールしてもらえる可能性は高い。
- AxAx、JxJx
  - こちらが負けているレンジ。**バリューターゲットではない。**
- Jx2x、8h4x、4x2x
  - 相手が一貫して相手が強い主張をしている以上、この時点のレンジに含まれているとは考えづらく**バリューターゲットとしては可能性が低い。**

相手のレンジにある強いバリューターゲットと弱いバリューターゲットの割合に沿ってオーバーポットのベットをするか小さいベットをするか選択するとよい。例えば（細かな計算を省略して）相手のレンジのバリューターゲットのうち、強いものが全体の 2 割、弱いものが 8 割ほどであるとき、10 回に 2 回ほどの割合でオーバーポットのベット、10 回に 8 回ほどの割合で小さなバリューベットをするとよい。

## ショーダウンバリュー

例えば KhJd2hAc7h のようなウェットなボードで自分が As9d を持ってオープンしていたとき、相手がドローハンドを完成させている可能性も充分あるが、同じように完成させてない可能性も大きくある。また、KxXx や JxXx のようなワンペアでしかないかもしれない。こういうケースでは:

- **自分からベットすると、相手にブラフのチャンスを与えることになる。**
  - 自分は Ah を持っていないので、相手のレンジの AhXh を否定できない。また Tx9x のようなハンドもブラフとして説得力があり否定できない。
- 自分からチェックしても相手はブラフをしてくるかもしれない。**しかし自分からベットして相手にレイズされるよりもずっと安い出費で済む。**

マージナルながらある程度強いハンドはショーダウンに持ち込むことで一定の利益があるので、リスクを抑えて利益を確保するのを優先した方がいい。そのときのバリューを**ショーダウンバリュー**という。
