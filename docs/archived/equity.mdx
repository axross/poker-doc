# エクイティ

**エクイティ**とはハンドの価値の具体的な期待値のこと。期待値（EV）と表されることもある。勝率とショーダウン時のポットサイズを掛け合わせて計算できる。

例として、勝率が 70％で、勝った時のポットが 200BB だと見込める場合、エクイティは 140BB となる。言い換えると、200BB のポットを勝率 70％のプレイヤーと勝率 30％のプレイヤーとで分け合っている形になる。

勝率の高いハンドが常にエクイティの高いハンドになるとは限らない。「AxAx は少し稼ぐか、大きく損するかである」はこのエクイティの不思議を示す代表的なものである。AxAx は相手のハンドレンジの Ax を 2 枚ブロックしていることになるので、フロップのボードが AxXxXx のようになった時に相手が Ax を持っている可能性が低い。結果として AxAx はそのままペアやツーペアとしてリバーを迎えることが多く、自分からポットを大きくするほどリスクが大きくなっていく。反面、ウェットなボードで AxAx が敗北する時はポットが大きくなっていることが多く、損失も大きい。

いくつか他の例をあげてみる。

| ハンド | エクイティ | 勝利時のポットサイズ | 勝率 |
| :----: | :--------: | :------------------: | :--: |
|  AxAx  |     中     |        小さい        | 高い |
|  7s6s  |     低     |        大きい        | 低い |

## フォールドエクイティ

- KsQs9s4s のようなウェットなボードで AdKh を持った Alice と QhJh を持った Bob がいて、ターンの 50BB のポットに Alice が 20BB のベットをしたとする。Bob がコールした後、リバーでの期待値はそれぞれ Alice が $71.59BB = \frac{35}{44}  \times 90$ 、Bob が $18.41BB = \frac{9}{44}  \times 90$ である
- Bob がフラッシュを嫌ってフォールドしたとすると、それはそのまま 18.41BB の期待値を放棄したことになり、当然 Alice はポットにある 70BB をそのまま受け取る形になる
- 同じシチュエーションで Alice が毎回この 20BB のベットを行ったときに Bob が 50％の割合でフォールドすると、Alice のこのアクションには平均して $7.16BB = \frac{9}{44} \times 70 \times 0.5$ のフォールドエクイティがあると考えられる
- つまり、Alice のこのアクションは 50％の割合で本来の期待値（=Alice が 71.59BB、Bob が 18.41BB）、50％の割合で 90BB を得ることになる
