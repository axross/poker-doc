# アウツとオッズ

## アウツ

例えば 4sKd7dTs のボードで Js9s を持っている時、

- AsKsQs8s7s6s5s3s2s のいずれかでフラッシュになる。
- QhQdQc8h8d8c のいずれかでストレートになる。

この引いた時にメイドハンドになるカードをアウツという。この例ではフラッシュかストレートになるために合計で 15 アウツある。

## ポットオッズ

相手にいくらかの額をベットされても、この 15 アウツを加味するとコールした方が利益的になることがある。

例えば、ポットが 100BB あるところに相手が 20BB ベットしたとき、このベットにコールすると最終ポットは 140BB（$100 + 20 + 20$）になる。20BB を差し出すことで 140BB を得られる機会が得られると言い換えられるので、

- 勝率が 10％なら期待値は **14BB**（$140 * 0.1$）
- 勝率が 25％なら期待値は **37.5BB**（$140 * 0.25$）
- 勝率が 50％なら期待値は **70BB**（$140 * 0.5$）

という形で、一定の勝率があれば期待値がコールに必要な額を上回ることになる。

ヘッズアップ、自分がインポジションのシチュエーション、 4sKd7dTs のボードで Js9s を持っているとする。現状自分が負けている可能性が高くとも、15 アウツで逆転できる。既に明らかになっているカードを除いたデッキの残り枚数は 45 枚なので、リバーで逆転する確率は$\frac{15}{45}$である。相手のベット額で考えてみると、

- 相手のベット額が 20BB（ポットの 20％）のとき、**期待値は 46.7BB**（$(100 + 20 + 20) * \frac{15}{45}$）
- 相手のベット額が 60BB（ポットの 60％）のとき、**期待値は 73.3BB**（$(100 + 60 + 60) * \frac{15}{45}$）
- 相手のベット額が 100BB（ポットの 100％）のとき、**期待値は 100BB**（$(100 + 100 + 100) * \frac{15}{45}$）

といった形で、**100BB（ポットの 100％）までのベットにはコールが正当化される。**この状態を「オッズが合う」と表現することが多い。

## 合わないアウツを押し付ける

- 参加のための自身の必要勝率と、相手に押し付けたい必要勝率のプレッシャーのバランスによって最適なベットサイズを求めることができる
  - 自身の要求勝率
    - ポットを $P$ 、ベット額を $B$ とした時、 $\frac{B}{P + B + B}$ で求められる。コールする時は $\frac{B}{P + B}$
    - 1.5BB のポットに 3BB でオープンする場合、要求勝率は $46.2\% = \frac{3}{1.5 + 3 + 2}$ となる
      - 自分の後に BB のみがコールするシチュエーションがポストフロップに進む中で最低のポットサイズとなるため分母は $1.5+3+2$
      - この勝率はフォールドエクイティを加味していない
    - 4.5BB のポットに 3BB でコールする場合、要求勝率は $40\% = \frac{3}{4.5 + 3}$ となる
  - 相手に押し付けたい要求勝率
    - ポットを $P$ 、ベット額を $B$ とした時、 $\frac{B}{P + B + B}$ で求められる
    - 1.5BB のポットに 3BB でオープンする場合、ブラインドポジションでない相手に押し付けられる要求勝率は $40\% = \frac{3}{1.5 + 3 + 3}$ となる
      - その後のセカンドコーラーの要求勝率は $28.6\% = \frac{3}{1.5 + 3 + 3 + 3}$ となる。あるいは、セカンドコーラーの出現が確実な場合のファーストコーラーの要求勝率も同じである
    - 1.5BB のポットに 5BB でオープンする場合、ブラインドポジションでない相手に押し付けられる要求勝率は $43.5\% = \frac{5}{1.5 + 5 + 5}$ となる
    - 1.5BB のポットに 10BB でオープンする場合、ブラインドポジションでない相手に押し付けられる要求勝率は $46.5\% = \frac{10}{1.5 + 10 + 10}$ となる
- 実際のところ、ベットサイズが相手に押し付けたい要求勝率として数学的なプレッシャーにはなりにくい。 $P \ge 0$ なので常に $\frac{B}{P + B + B} \le 0.5$ であり、2.5BB ほどで充分と言える（ここ続きを書く）
- ベットサイズが直接影響するのはエフェクティブスタックとの割合である。たとえば同じ 10BB でのベットでも、エフェクティブスタックが 200BB の時と 30BB の時とではその後のゲームツリーが違う。
  - エフェクティブスタックが 30BB では、リアクションとしてオールインかフォールドの方が利益的になりやすい
  - 間接的にフォールドエクイティが変わるとも言える
