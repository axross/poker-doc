# ポジション

## ポジションの優位性

インポジションのプレイヤーはアウトオブポジションのプレイヤーよりも利益が大きく損失が小さくなりやすい。各ストリートでインポジションのプレイヤーは相手の動きを見てからチェックとレイズを選択できるからである。

## ストラドル

テーブルによってはストラドルが許可されていて、ハンドが配られる前にビッグブラインドの倍額を払うことでプリフロップでの行動順を最後にすることができる。これはその後のゲームにいくつかの変化をもたらす:

- **SPR が半分以下に低下する。**ストラドル額が実質的なブラインド額になるので、ポット額が倍以上に大きくなる。
- **VPIP が上昇する。**誰もオープンしなかった場合はチェックが可能で、別の考え方をするとハンドに関係なくオープンしているのと同じ状況になる。
- **アグレッションが低下する。**ビッグブラインドのアグレッションが低いのと同じ理由。

こういった理由から、ストラドルはゲームのボラティリティを上昇させる。テーブルに自分よりも上手いプレイヤーが多い場合は活用した方がいい。自分よりも下手なプレイヤーが多い場合はあまり利益的ではない。
