## ハンドと強さのリスト

| ハンドの名前         |    説明    | 説明                                                                        | できやすさ |
| :------------------- | :--------: | :-------------------------------------------------------------------------- | :--------: |
| ストレートフラッシュ | QhJhTh9h8h | ストレートとフラッシュが同時に成立している 5 枚                             |            |
| クアッズ             | 6x6x6x6xXx | 5 枚の中に同じ数字のカードが 4 枚ある                                       |            |
| フルハウス           | JxJxJx2x2x | 別々のトリップスとペアで構成される 5 枚                                     |            |
| フラッシュ           | XhXhXhXhXh | 同じスート（柄）の 5 枚                                                     |            |
| ストレート           | 5x4x3x2xAx | 連番になっている 5 枚<br/>A は 1（2 の前）としても 14（K の後）としても扱う |            |
| トリップス           | QxQxQxXxXx | 5 枚の中に同じ数字のカードが 3 枚ある                                       |            |
| ツーペア             | AxAx8x8xXx | 5 枚の中に別々のペアが 2 つある                                             |            |
| ペア                 | 7x7xXxXxXx | 同じ数字のカードが 2 枚ある                                                 |            |
| ハイカード           | XxXxXxXxXx | 上記のどれでもない                                                          |            |
