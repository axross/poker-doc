# ハンドレンジ

相手がどんなカードを持っているか 1 点読みすることは不可能である。代わりに**ハンドレンジ**という持ちうるハンドの組み合わせで相手のカードを推測していく。

一般的な傾向として、強いハンドはレイズ率や参加率が高く、弱いハンドはレイズ率も参加率も低い。

あるプレイヤーがプリフロップで 3BET をした場合、そのプレイヤーのハンドは比較的に強い可能性が高い。AxAx や KxKx のようなハンドはもちろん、AxQx や QhJh のようなハンドも考えられる。

そしてこの 3BET にコールしたプレイヤーがいたとき、そのプレイヤーのハンドは 3BET したプレイヤーよりも弱い傾向がある。AxAx や KxKx のようなハンドは 4BET（レイズ）している可能性が高いためである。

この時点で、次のようなことが言える:

- 3BET したプレイヤーには
  - 強いハンドがある可能性が高い。上限はなく AxAx を持っている可能性すら否定できない。
  - 弱いハンドがある可能性が低い。7x2x のようなハンドを持っているとは考えづらい。
- 3BET にコールしたプレイヤーには
  - まあまあ強いハンドはあるが、一定の上限がある。強くとも QxQx や JxJx あたりだろうと考えられる。
  - 弱いハンドがある可能性は否定できない。とはいえ、Ah5h、7s6s、8x8x といった一定以上のエクイティのあるハンドだと考えられる。

## ハンドの推測

これらの情報をポストフロップで活用する。説明を簡単にするために 3BET したプレイヤー（以降 Alice と呼ぶ）とコールしたプレイヤー（以降 Bob と呼ぶ）のハンドレンジを次のように断定して考える（あなたはこの 2 人と何万回もプレイしていて傾向を知っている）。

- Alice（3BET したプレイヤー）のハンドレンジ
  - AxAx-2x2x
  - A=K=-A=9=、A=5=-A=2=
  - K=Q=-K=J=、KxQx-KxJx
  - Q=J=-Q=J=
- Bob（コールしたプレイヤー）のハンドレンジ
  - QxQx-2x2x
    - （AxAx、KxKx は 4BET しているはずなのであり得ない）
  - A=Q=-A=2=、AxKx-AxTx
  - K=Q=-K=T=、KxQx-KxJx
  - Q=J=-Q=T=、QxJx
  - J=T=-5=4=、J=9=-6=4= などのスーテッドコネクター

フロップが Jd8h5h となり、Bob はチェック。Alice はポットの 50％をベットし、Bob はそれにコールした。Bob はペアができたのかもしれないし、ドローハンドになったのかもしれない。あるいはツーオーバーなのかもしれない。何を持っているか 1 点に読み切ることはできないが、少なくともコールできるハンドがあるということはわかる（Alice のベットはコンティニュエーションベットで、何を持っているかまだわからない）。

Bob のハンドレンジの中で、コールできそうなハンドを洗い出してみる。ここで重要なのは、**プリフロップまでに存在しなかったハンドがフロップのハンドレンジに急に出現したりしないということである。**Bob のハンドレンジに Jx8x や Jx5x がなかった以上、**Bob にはツーペアができていないということがわかる。**

- Bob のハンドレンジ（フロップでのコール後の時点）
  - QxQx-2x2x
  - A=Q=-A=J=、A=5=、A=2=、AhTh-Ah2h、AxKx-AxJx
  - K=Q=-K=J=、KxQx-KxJx
  - Q=J=、QhTh、QxJx
  - J=T=-9=8=、J=9=-8=6=、JhTh、Th9h、7h6h-5h4h、Jh9h、9h7h、7h5h、6h4h など

現時点で Jx か 8x のワンペア以上ができているか、フラッシュドローやガットショットドローなどのドローハンドという条件で絞ってみた。存在が否定できるハンドがいくつか出てきたり、スートが特定的になったものがあることがわかる。

ターンでは Qd が開き、ボードは Jd8h5hQd になった。Bob はチェックし、Alice は引き続きポットの 50％をベット、Bob はコールした。

Alice のターンでのベットはフロップのベットよりもブラフの可能性が低くなっている。なぜなら Alice のフロップのベットがブラフだった場合、Bob のコールは怖く見えるからだ。Alice の「降りろ」というメッセージに対して Bob は「いいや降りない」と返事したからだ。それでも Alice はターンでもう一度ベットした。Bob のコールからは「お前を倒してやるぞ」という強いメッセージが読み取れる。

そう考えると、弱いワンペアやキッカーの弱い Jx ヒット、アウツの少ないドローハンドが否定できることがわかる。Bob のハンドレンジはさらに絞られた。ここでも、**フロップまでで存在が否定されていたハンドが急に顔を覗かせたりはしない。**

- Bob のハンドレンジ（ターンでのコール後の時点）
  - QxQx、JxJx、8x8x、2x2x
  - A=Q=、A=J=、Ah8h、Ah2h、AxJx
  - K=Q=、K=J=、KxQx、KxJx
  - Q=J=、QhTh、QxJx
  - J=9=、T=9=、T=8=、JhTh、Th9h、7h6h-5h4h、Jh9h、9h7h、7h5h、6h4h など

リバーを見てみよう。リバーでは c が開き、ボードは Jd8h2hQd2c になった。ここで Bob がポットの 60％をベットし、Alice はコールするか悩むことになった。Bob のベットが何を意味しているか考える前に、Bob のハンドレンジをもう一度洗い出してみる。この Bob のベットから言えることは、Bob にとって 2c はまったく困らないカードか、むしろ嬉しいカードかのどちらかだということだ。

- Bob のハンドレンジ（リバーでのベット後の時点）
  - QxQx、JxJx、8x8x、2x2x
  - A=Q=、A=J=、Ah8h、Ah2h、AxJx
  - K=Q=、K=J=、KxQx、KxJx
  - Q=J=、QhTh、QxJx
  - J=9=、T=9=、T=8=、JhTh、Th9h、7h6h-5h4h、Jh9h、9h7h、7h5h、6h4h など

## スターティングハンドのマジックナンバー

- 全スターティングハンドの組み合わせ - $1326 = {}_{52}\mathrm{C}_{2}$通り
  - 全ポケットペアの組み合わせ - $78 = 13 \times {}_{4}\mathrm{C}_{2}$通り
  - 全スーテッドハンドの組み合わせ - $312 = 4\times\sum_{i=1}^{12}$通り
    - 全スーテッドコネクターの組み合わせ - $48 = 4\times12$通り
    - 特定のスーテッドハンドの組み合わせ - $4$通り
  - 全オフスートハンドの組み合わせ - $936 = P(4)P(3)\times\sum_{i=1}^{12}$通り
    - 特定のオフスートハンドの組み合わせ - $12 = P(4)P(3)$通り

## ハンドレンジ

- ハンドレンジは 2 枚のカードとその蓋然性で構成される
  - AsKh のような 2 枚のカードがハンドレンジ内にどれくらいの確率で存在しうるか。AsKh が必ず存在する場合は*AsKh(100%)*と考えられ、コールする場合とフォールドする場合が等しくあるなどして 50％の確率で存在しうるのであれば*AsKh(50%)*である
  - こういった形で存在しうる上限から下限までハンドを網羅したものが完全なハンドレンジである
    - 例として、あるプレイヤーのプリフロップでのコールによってハンドレンジの推測が進み、AA と KK を確実に否定できるのであればそのハンドレンジから AA と KK が消え、上限は*QQ(14%)*のようになる
      - 仮の数値だが*14%*としている理由として、AA と KK を否定できるハンドレンジでの QQ の蓋然性が 100％であるとは考えにくい
  - この蓋然性の概念によってハンドレンジは三次元的になり計算コストは増えるが正確性が増す
- このハンドレンジの考え方は相手のハンドを推測する時にも使えるし、同様に相手から見た自分のハンドレンジを主張する時にも活用できる
  - 自分が相手のハンドレンジを確定することができないように、相手から自分のハンドレンジを確定することはできない
  - 自分の持っている 2 枚のカードを元に一人称的にプレイしてしまうと、本来取れる（あるいは「取るべき」）アクションを取れなくなる（ブラフ）
- 自分の持っている 2 枚のカードはショーダウンまであまりゲームに関係していない。ある 2 枚のカードが持つ価値はショーダウンすることで初めて評価される

## ゲームツリーの経路におけるハンドレンジの変化

- あるプレイヤーのスターティングハンドレンジについて確実な把握をしているとする。プレイヤーがアクションを行うごとに、高い蓋然性で否定できるハンドが増えていく。例えば、レイズとコールで用いられるハンドが同じなんてことはない
- 言い換えると、ハンドレンジはゲームツリーの経路上で漸進的に変化していく。頂点ごとに異なるハンドレンジ（変化後のスナップショット）を持つ
  - 経路 $R$ 上に連続する頂点 $V^n$ と頂点 $V^{n+1}$ で、頂点 $V^n$ で否定されていたハンド（0%の蓋然性）の存在が頂点 $V^{n+1}$ で再評価される（>0%の蓋然性）などということはない。そうとしか考えられないアクションを目撃したとしたら、それはブラフの発覚である
  - 反面教師的に考えると、それまでに否定されていたハンドの存在が復活するようなブラフを行ってはならない。言い換えると、頂点 $V^n$ でのハンドレンジは常に頂点 $V^{n-1}$ のハンドレンジより狭くなければならない（アクションの一貫性）。

## フロップでハンドは成長したか？

- ポケットペア以外のハンドがフロップでワンペアになる確率は $32.43\% = \frac{{}_{50}C_3-{}_{44}C_3}{{}_{50}C_3}$ である
  - 裏を返すと、自分も相手も残りの 67.57%ほどの確率でハンドをミスしていることになる（この 2 つの事象に関連はないとする）
  - この「67.57％で相手がミスする」という事実は、ファーストアクションでフォールドエクイティとしてハンドの価値に換算できる
- アグレッサーのハンドレンジはハイランクに偏りがあり、コーラーのハンドレンジにはミドルからローランクへの偏りがある
  - また特定の状況を除いては、コーラーのハンドレンジにはアグレッサーに比べてスーテッドハンドが多いという偏りもある
    - オフスートには A9o のような、オープンはできるがコールが難しいハンドが多いため
- フロップにハイランクのカードが含まれていればアグレッサーにとって有利であり、ミドルランクのカードで大部分が構成されていればコーラーにとって有利であると言える
  - アグレッサーにとってはハイランクのカードが「含まれて」いればよいのに対し、コーラーにとってはフロップがミドルランクで「大部分が構成されて」いなければ有利とは言えない。例えば A86 や KJ2 のようなフロップではコーラーは自身の有利に確証を持てない
