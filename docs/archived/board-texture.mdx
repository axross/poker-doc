# ボードテクスチャー

ボードテクスチャーとはコミュニティカードの組み合わせの特徴のこと。

## ウェットなボード

ストレート以上のハンドができやすいボードを**ウェットなボード**と言う。例えば:

- Ks8s3s - フラッシュになりやすく、既に完成しているプレイヤーもいるかもしれない。
- JcTd9s - ストレートになりやすく、既に完成しているプレイヤーもいるかもしれない。
- 8h6d3h - フラッシュにもストレートにもなりやすい。既に完成しているプレイヤーはいないが、フラッシュドローになっていたり、オープンエンドやダブルベリーバスターになっているプレイヤーがいるかもしれない。

ウェットなボードではコンティニエーションベットのフォールドエクイティが低い。

## ドライなボード

ストレートやフラッシュの警戒をしなくていいボードを**ドライなボード**と言う。Ks8d3c など。

ドライなボードはワンペアやツーペアなどのヒットが勝因になりやすく、ポットが大きくなりにくい。ドローが少なく、コンティニエーションベットのフォールドエクイティが高い。

## ハイボード

AxKxQxJx のいずれかが含まれているボードをハイボードと言う。

オープンしたプレイヤーのハンドレンジにはハイカードが比較的多く含まれているので、オープンしたプレイヤーにとって有利なボードと言える。

## ローボード

Tx9x8x7x6x5x4x3x2x のみで構成されているボードをローボードと言う。

コールしたプレイヤーのハンドレンジにはローカードが比較的多く含まれているので、コーラーにとって有利なボードと言える。
