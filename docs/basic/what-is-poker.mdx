# ポーカーとはどんなゲームか

## ポーカーは論理的思考と選択のゲーム

ポーカーは論理的で合理的な選択を取り続けるゲームです。ブラフを成功させるためにはプリフロップからそれまでのアクションに論理的一貫性がなければ説得力がなく、相手にブラフを看破されてしまいます。逆にブラフをまったくしないプレイヤーはバリューベットをコールしてもらえません。逆の見方をすると、バリューベットによって利益を出すためにも論理的なプレイが求められるということです。

## ポーカーは将棋や囲碁に近いマインドゲーム

ポーカーには運要素があり、お金をやり取りするギャンブルです。しかしながら、ポーカーを数学的目線で分析していくと将棋や囲碁に近いマインドゲームであることが明らかになっていきます。将棋や囲碁で次の一手・そしてその後の相手のとりうる手を考えるのと同じように、ポーカーもどのアクションをとっていくかを考える上で次の相手のアクションの可能性を考えながらプレイしていきます。事実、多くの将棋の解析ソフトとポーカーの解析ソフト（ソルバー）はゲームツリーを構築して最大効果になる探索を行うという非常に近いアプローチをとっています。

:::note

余談ですが、将棋よりもポーカーの方が計算空間が広く複雑だと言われていて、一般的にポーカーのソルバーの方が多くの計算時間を必要とします。

:::

## ポーカーは心理戦ではない

何度もプレイしたことのある人とプレイする場合はその人がどんなプレイをするか推測できることがありますが、カジノのキャッシュゲームやマルチテーブルのトーナメントでは知らない人と戦うことも少なくありません。また、相手がゲーム理論的な戦略に基づいて行動している場合、心理的なアドバンテージを取ることが非常に難しくなります（詳しくは[中級編](../intermediate/what-is-poker)をご覧ください）。

「相手の裏をかく」「さらにその裏をかく」という考え方はポーカーのプレイが単純（大きなブラフベット or 小さなバリューベット など）な場合には有効ですが、ある程度のセオリーを知っているプレイヤーには通用しません。
