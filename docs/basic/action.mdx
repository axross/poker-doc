# アクション

## なぜリンプしてはいけないのか

ポーカーを勉強しているとよく「リンプするな」と耳にすることがあると思います。何故なのか考えてみます。

### 強いハンドで稼げない

まず一番のデメリットは、AsAh を持ってようと KdKc を持ってようと大きなポットを得るチャンスを失うということです。

自分のハンドが強い時にポットの 20 倍のような大きなベットをしても誰からもコールしてもらえません。結局のところ、それまでに積み上げてきたポットサイズに比例してベットサイズが決まっていきます。リンプして参加するということは、強いハンドでの利益を損ねてしまうプレイなのです。

### 弱いハンドにチャンスを与えてしまう

リンプして参加すると、他のプレイヤーの参加オッズが良くなります。結果として、多くのプレイヤーからコールされ、大人数でフロップを見にいくことになります。

AsKs のような強いハンドでも、6 人でプレイすると勝率は 28%弱にまで落ち込みます。きちんとレイズして弱いハンドのプレイヤーを振り下ろし、参加人数をコントロールして 3 人でプレイできるようにすれば依然 41%ほどの勝率でプレイできます。

参加人数が多ければ多いほどリスクは高まります。（強いことを隠して）AsAh でリンプして 5 人でフロップを開き、8d5d4d のようなボードが開いた時には最悪です。フラッシュやストレートが完成しているプレイヤーがいるかもしれませんし、そうでなくともドローハンドを持っているプレイヤーにターンやリバーで完成させられてしまうかもしれません。他のプレイヤーを降ろそうとビクビクしながらベットしても手遅れです。だったらプリフロップできちんとレイズしておけば...。

### 弱いハンドでリンプしても結局稼げない

逆に、弱いハンドで参加するためにリンプしてみたらどうでしょう。一見良いアイデアに思えますが、弱いハンドでは弱い役しかできあがりません。Js5h のようなハンドでプレイして 5x のワンペアができても、それがベストハンドだという確証を持てません。

また、運よく Js4d5h のようなフロップでツーペアができたとしましょう。でもその時にポットにあるのは人数分のブラインドだけです。このようなフロップでのツーペアの完成はおよそ $$\frac{1}{20}$$ で起こりますが、残りの $$\frac{19}{20}$$ は取るに足らない弱い役になるということです。果たしてその 20 倍のオッズに見合うような儲けを得られるでしょうか？
