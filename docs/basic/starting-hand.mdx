# スターティングハンド

ポーカーで最初に配られる 2 枚のカードをハンドと言い、どんなハンドをプレイするか考えるときにはスターティングハンドと表現します。

## どういうハンドでプレイするか

### 強いハンドと弱いハンド

強いものから弱いものまで様々なハンドがあります。同じランクの 2 枚のカードで構成されるハンド「**ポケットペア**」は最初からワンペアができていることが確定していて強いです。反面、バラバラの低いランクで構成されているハンドは弱いです。

最も強いハンドは AsAh で、最も弱いハンドは 7x2x と 3x2x です。最も弱いハンドが 2 種類あるのは相手にする人数によって「純粋なランクの強さ」と「ストレートへのなりやすさ」のどちらが大切かが分かれるためです。一般的にヘッズアップの場合は 7x2x の方が強く、2 人以上を相手にする場合は 3x2x の方が強いです。

### 期待値の高いハンドと低いハンド

期待値とはハンドをショーダウンまでプレイした時に得られる平均のポットサイズです。通常、強いハンドはプラスの期待値、弱いハンドはマイナスの期待値になります。

ただし、ハンドの強さと期待値がそのまま相関するかというとそうではありません。例として、Kh9h と JdTd では勝率と期待値がおよそ直感と異なる結果になります。

| ハンド | 勝率（ヘッズアップ - 10max） |   期待値    |
| :----- | :--------------------------: | :---------: |
| Kh9h   |       **60.0 - 14.8%**       |   0.09BB    |
| JdTd   |        57.5% - 16.5%         | **0.15 BB** |

これは言い換えると「強いハンドが稼げるハンドというわけではない」ということになります。ポーカーのよくある金言として次のようなものがあります。

> AsAh は小さく勝つか、大きく失うかのどちらかである。

自分が AsAh を持っている時点で Ax を 2 枚ブロックしてしまっているので、相手が Ax を持っている可能性が低いのです。結果として、AsAh はプリフロップのレイズで終わってしまい、ブラインドぶんしか稼げないということがよく起こります。

フロップが開いたとしても、AsAh が相手にするのはドローハンドが多く、ドローハンドからは大きなバリューを見込めません。ドローハンドがショーダウンするのはフラッシュやストレートが完成した時ですが、このとき AsAh でフォールドする決断をするのは非常に難しいのです。

## ポジションとの関係

どういったハンドで参加するか大まかな目安として、勝率（または期待値）の上位 $$\frac{1}{N} \times 1.2$$ のハンドでプレイするのが効果的とされています。この $$N$$ はフォールドせずに残っているプレイヤーの人数です。

6-max のテーブルで自分が UTG にいる時は上位 $$\frac{1}{6} \times 1.2$$ のハンドでプレイするのが好ましいです。また、自分が BTN にいて自分の前のプレイヤーがすべてフォールドしていたなら残りの人数は自分を含めて 3 人です。この場合は上位 $$\frac{1}{3}\times 1.2$$ のハンドで充分だということになります。

## 周りに参加率の高いプレイヤーが多い時

参加率の高いプレイヤーと参加率の低いプレイヤーが同じテーブル上にいた時、**少数派になる選択をした方が利益的**になります。

参加率の高いプレイヤーが周りに多い場合は、自分の参加率を周りより低くします。参加率が高いということはスターティングハンドが弱い傾向にあります。自分の参加率を低くすることで、強いハンドで弱いハンドと戦えるようになります。

逆に周りのプレイヤーの参加率が低いと感じた場合は、自分の参加率を高くします。参加率が低いプレイヤーのハンドは強いですが、それだけフォールドするということでもあります。参加率の低いプレイヤーが多いテーブルではレイズによってポットをスティールできることが多いです。
