# アウツとオッズ

## アウツ

アウツとは、特定のメイドハンドができるのに必要なカードのことを言います。例えば 4sKd7dTs のボードで自分は 9s6s を持っている時、

- AsKsQsJs8s7s5s3s2s のいずれかでフラッシュ
- 8h8d8c のいずれかでストレート

になります。この例ではフラッシュかストレートになるために合計で 12 アウツあることになります。

## ポットオッズ

相手にいくらかの額をベットされても、アウツを加味するとコールした方が利益的になることがあります。例えば、100BB のポットに相手が 50BB のベットをしてきたとき、このベットにコールすると最終ポットは 200BB（$$100 + 50 + 50$$）になり、50BB を差し出すことで 200BB を得られる機会あると言い換えられます。

これを踏まえて、自分のハンドの勝率ごとに期待値を計算してみると次のようになります。

| ハンドの勝率 |                  期待値                   |
| :----------: | :---------------------------------------: |
|     1/6      | **33.33BB**（$$200 \times \frac{1}{6}$$） |
|     25%      |        **50BB**（$$200 * 0.25$$）         |
|     1/3      | **66.66BB**（$$200 \times \frac{1}{3}$$） |

一定の勝率があれば期待値がコール必要額を上回ることになります。この場合のコール額は 50BB なので、勝率が 25％以上ならばコールした方が利益的だと言えます。

### コールの正当化（オッズが合う）

4sKd7dTs のボードで自分は 9s6s を持っている時、12 アウツあるという話をしました。自分のハンドの 2 枚とボードの 4 枚を除くと残りのデッキの枚数は 44 枚なので、リバーで 12 アウツのいずれかを引き当てる確率は $$\frac{12}{44}$$ です。これは約 27.27%ですので、コールする方が利益的です。

このアウツとポットオッズの兼ね合いでコール以上のアクションをとった方が利益的になることを **オッズが合う** と表現します。

### ドローハンドの正当コール上限の一覧表

自分のアウツから考えてみると、それぞれいくらまでのベットに対応できるか計算できます。以下の表はドローの種類ごとのアウツの数と、コールが正当化される（利益的になる）上限のベットサイズの一覧です。

| ドローの種類                            | アウツの数（必要勝率） | コールが正当化される上限<br/>（ベット前のポットの X％） |

| オープンエンド<br/>ダブルベリーバスター |     8 枚（17.39%）     |                         26.67％                         |

### 合わないアウツを押し付ける

ボードやこれまでの相手のアクションから相手のハンドが何らかのドローハンドであることが推測できるとき、相手にとってコールが正当化されるベットサイズの上限以上の額をベットすることで、相手にとってコールするのが利益的ではない状況を作り出すことができます。
